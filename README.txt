Для запуска данного проекта Вам необходимо:
1.Клонировать репозиторий с GitHub на Ваш компьютер.
2.Открыть проект в любом IDE.
3.Запустить
В данном проекте не написаны тесты так как они не нужны, потому что протестировать можно метод main и метод begin,
что не требуеться, другие же методы имеют модификатор доступа private, что не позволит полуить к ним доступ из другого
класа(можно конечно но не стоит), а изменить их модификаторы доступа не возможно так как тогда будет нарушены стандарты
Java кода, а именно правило Инкапсуляции.
